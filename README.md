# Number Utilities Project


## Функціонал
- `fibonacci(n)` — обчислення чисел Фібоначчі з мемоїзацією.
- `factorial(n)` — рекурсивне обчислення факторіала.
- `isPrime(n)` — перевірка числа на простоту.
- `range(start, end, step)` — генерація масиву чисел.
- `clamp(value, min, max)` — обмеження числа діапазоном.
- `memoize(fn)` — універсальна функція для кешування результатів.
