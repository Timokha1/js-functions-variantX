# js-functions-variantX


## Функціональні можливості
1.  **`isPalindrome(str)`**: Перевірка рядка на паліндром. Автоматично ігнорує регістр, пробіли та неалфавітні символи.
2.  **`countWords(str)`**: Аналіз тексту. Повертає загальну кількість слів, кількість унікальних одиниць та найбільш вживане слово.
3.  **`truncate(str, max, suffix)`**: Безпечне обрізання рядка до заданої довжини із додаванням суфікса (за замовчуванням `...`).
4.  **`reverseWords(str)`**: Реверс порядку слів у реченні зі збереженням структури слів.
5.  **`memoize(fn)`**: Функція вищого порядку для кешування результатів обчислень на основі вхідних аргументів.
